HEXO博客搭建教程（一）

2023/12/12 hexo 教程

Hexo是一个博客框架，使用Markdown的语法来解析文章。可以快速地生成一个静态网页。

1.安装环境

1.1 Git和Node.js

Hexo文档

Git 官网

Node.js官网

The MacPorts Project – Home

mac安装Git需要先使用MacPorts

$ sudo port install git

完成后在控制台分别输入Git –version
Node -v
git -v

1.2 博客本地环境

Hexo官网 hexo是一款台湾人开发的博客框架，对中文友好。

安装前，在本地新建一个文件夹

进入新文件夹，鼠标右键

进入hexo官网，输入以下指令。

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

已经成功建立了一个本地博客。

2. 小试牛刀

2.1 新建文章

hexo new "瓦达西の第一篇article"

hexo g
hexo s

2.2 挑选主题

还是进入hexo官网。

挑选一个好看的主题。

然后根据主题的教程安装。~~~（需要小白有很强的决心）~~~
